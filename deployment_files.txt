# Telegram Casino Bot

A comprehensive Telegram casino bot with emoji games.

## Features
- Emoji games: dice, basketball, bowling, darts, soccer, slots, blackjack, mines
- Admin panel with balance management
- User tipping system
- PostgreSQL database integration

## Deployment
1. Set environment variables: BOT_TOKEN, DATABASE_URL
2. Run: pip install -r requirements.txt
3. Run: python main.py

## Admin Commands
- /addbalance - Add balance to users (Owner & Developer only)
- /admin - Admin panel access

## Game Commands
- /dice, /basketball, /bowling, /darts, /soccer, /slots, /blackjack, /mines
- /tip - Send money to other users
- /balance - Check balance
- /help - Show help
